# After a Debian GNU/Linux WSL fresh install
## Update and upgrade

```bash
sudo apt update
sudo apt update -y
```

## Install git
```bash
sudo apt install git
```

## Clone this repo
```bash
git clone https://github.com/pietrodito/debian-wsl-setup.git ~/Comp/debain-wsl-setup
```

## Run installers
```bash
cd ~/Comp/debain-wsl-setup
./run-installers.sh
```
